What each package does:
streamlit: Powering your web-based frontend/dashboard.

easyocr: The "eyes" of your system that read text from images and PDFs.

pymupdf: (Installed as fitz) Used for high-speed PDF parsing and structural analysis (checking for images/tables).

groq: The client for the ultra-fast LLM API that performs the "Audit."

python-dotenv: Securely loads your Groq API key from a .env file.

fpdf: Used by the dummy data script to generate professional-looking insurance PDFs.

faker: Generates realistic names, addresses, and policy numbers for testing.

python-magic: Helps identify file types to ensure the user is actually uploading documents.
